Contribute

Want to help develop the Open Source Edition of F#? Bring F# to OSX, Linux, Android and iOS? Fork the repository and get going!

Don't ask for permission to contribute, just fork the project on Github, commit your changes and send a pull request. You can even edit some files (like this one!) directly on GitHub and a pull request will be created automatically for you when you save.
